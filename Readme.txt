Steps to run the RAG model:
  a. Install the libraries and dependencies required.
  b. Log in to your google account using (gcloud auth login).
  c. Select the project where Vertex AI API is enabled.
  d. Create a service account in your google cloud account selected project and give it all the required permissions. Download the service account key in the .json format and place it in your root directory.
  e. Now you can run the the following application. Once the Gradio interface is started you can upload the documents and get the answer to your questions accordingly.
